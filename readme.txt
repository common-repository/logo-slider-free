== Installation ==
<strong>Using The WordPress Dashboard</strong>

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After activate plugin you will see "Logo Showcase" menu at left side on WordPress dashboard and find "Generate Shortcode" for generating logo showcase.<br />
<br />

<strong>Using FTP</strong>
1.Extract logo-slider-free.zip to your computer.
2.Upload the logo-slider-free directory to your wp-content/plugins directory.
3.After activate plugin you will see "Logo Showcase" menu at left side on WordPress dashboard and find "Generate Shortcode" for generating logo showcase.

<br />
= How to use short-code? =
When Logo Slider set-up done please copy & paste shortcode into your theme any where you want to display this Logo Showcase.
